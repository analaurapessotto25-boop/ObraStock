
  # Construir software conforme arquivos

  This is a code bundle for Construir software conforme arquivos. The original project is available at https://www.figma.com/design/eIP8IQoOTO8QI9x19y45qg/Construir-software-conforme-arquivos.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploy on Vercel

  If this repository is connected through GitHub, set the Vercel Root Directory to:

  `Construtora/Construir software conforme arquivos`

  The project includes `vercel.json` with:

  - Install Command: `npm install`
  - Build Command: `npm run build`
  - Output Directory: `dist`
